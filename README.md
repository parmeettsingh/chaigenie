ChaiGenie | Full-Stack Quick Commerce & Philanthropy Platform
Developed an innovative hybrid platform using Next.js 14 and Tailwind CSS that merges rapid-delivery quick commerce with a built-in donation ecosystem. Integrated a Genie AI Chatbot for interactive order management alongside a secure Razorpay Fintech integration to handle both commercial transactions and charitable contributions. Engineered a premium, state-driven "Genie" UI with Framer Motion to provide a seamless, high-conversion user experience across all devices.
To run this app- npm run dev
