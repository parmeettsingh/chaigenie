# 🧞‍♂️ ChaiGenie – Tap, Wish, Sip

ChaiGenie is a Next.js-powered **AI Quick Commerce PWA** designed to revolutionize how users order their favorite chai and snacks.

### 🚀 Features
- **GenieChat AI:** A smart, voice-enabled assistant that understands wishes in English and Hindi.
- **Context-Aware Recommendations:** Suggests the perfect "4 PM Masala Chai" based on time and order history.
- **PWA Ready:** Fully installable on mobile and desktop with offline manifest support.
- **Real-time Tracking:** Interactive timeline showing the "Genie Brewing" process.
- **Full-Stack:** Integrated with NextAuth for sessions and a custom cart context.

### 🛠️ Tech Stack
- **Frontend:** Next.js 14, Tailwind CSS, Framer Motion
- **Voice:** Web Speech API (Recognition & Synthesis)
- **State:** React Context API

### To run this app 
- npm run dev
